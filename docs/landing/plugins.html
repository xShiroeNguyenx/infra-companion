How to install

  1. Open Infra Companion (or download it — free, Windows · macOS · Linux).
  2. Menu ⋯ → 🧩 Plugins → tab 🛒 Marketplace.
  3. Click Install. That's it — the plugin's commands appear in the Command Palette instantly, no restart.

Updates show up in the same tab as an Update button when a newer version is published. Plugins run fault-isolated in a worker thread and can't read your vault or secrets — but they are trusted code: the catalog only lists signed entries.
